# Rocker Shiny

Este é o projeto Rocker Shiny. Um servidor Shiny para aplicações feitas em `R`.

## Setup Shiny

```bash
  docker build -t shiny-server .
```

> Certifique-se que você está dentro da pasta `rocker-shiny`.

## Como executar?

```bash
  docker-compose up -d
```

> Certifique-se que você está dentro da pasta `rocker-shiny`.
